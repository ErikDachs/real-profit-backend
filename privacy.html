Privacy Policy – Real Profit

Effective Date: 2026-03-04

Real Profit is a financial analytics application for Shopify merchants. The service analyzes order transaction data to calculate operational profit metrics for Shopify stores.

Data Access

With the merchant’s authorization, the application accesses limited store data through the Shopify API including:

  • Orders
  • Line items
  • Product identifiers
  • Transaction amounts
  • Refund information
  • Shipping charges

Purpose of Data Processing

The accessed data is used exclusively to calculate financial metrics for the merchant's store including:

  • Net profit per order
  • Contribution margin
  • Shipping subsidies
  • Payment fee impact
  • Profit trends

Customer Personal Data

Real Profit does not store or persist customer personal data.

The application does not store:

  • Customer names
  • Email addresses
  • Phone numbers
  • Shipping addresses
  • Billing addresses
  • Payment information

Order data is processed transiently for financial calculations and is not permanently stored.

Stored Data

The application stores only merchant configuration data including:

  • Shop domain
  • API access token
  • Cost model overrides
  • COGS overrides
  • Action plan state

Data Retention

Customer personal data is not retained by the application.

If a merchant uninstalls the application, all stored shop configuration data is deleted.

Compliance

The application implements Shopify data protection webhooks including:

  • app/uninstalled
  • shop/redact
  • customers/data_request
  • customers/redact
